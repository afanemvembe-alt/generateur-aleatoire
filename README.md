# Générateur de mots de passe personnalisés

Cette application Python permet de générer des mots de passe aléatoires et sécurisés de manière simple et intuitive, tout en intégrant des éléments personnalisés fournis par l'utilisateur.

L'utilisateur peut inclure des repères personnels dans son mot de passe, tels que :
* Un fruit préféré
* Un animal préféré
* Une date de naissance

Il est également possible de définir la longueur totale du mot de passe. L'application combine et mélange ces données personnelles avec des lettres, des chiffres et des caractères spéciaux afin de garantir un niveau de sécurité élevé et un résultat unique à chaque génération.

## Fonctionnalités principales

* **Interface graphique intuitive** : Développée avec Tkinter pour une prise en main rapide et claire.
* **Sécurisation personnalisée** : Génération de mots de passe forts combinant choix de l'utilisateur et variables aléatoires.
* **Copie rapide** : Intégration d'un bouton de copie directe du mot de passe généré dans le presse-papier.
* **Flexibilité** : Possibilité de tester instantanément différentes configurations, combinaisons et longueurs de caractères.

## Installation et utilisation

1. Assurez-vous d'avoir installé [Python 3](https://www.python.org/downloads/) sur votre système.
2. Téléchargez le fichier source `aleatoire.py`.
3. Lancez l'application depuis votre terminal avec la commande suivante :

```bash
python aleatoire.py
